This is the repository to host the project of ML for PdM course.
The project was done using [this dataset](https://github.com/boschresearch/CNC_Machining/).

To get started on the code, I suggest the following steps:

1. Cloning the original dataset repository, using the following command:

```
git clone git@github.com:boschresearch/CNC_Machining.git
```

2. Making sure that the dependencies from [Damavand](https://pydamavand.github.io/documentation/) are installed; to do so, execute the following command:

```
pip install -r damvand/requirements.txt
```

3. Open the `playground.ipynb` notebook and try running it; it should be executable, without any error.
